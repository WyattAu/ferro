# Ferro Licensing Strategy

## Overview

This document outlines the licensing strategy for Ferro.

## Open Source License

### Core License: AGPL-3.0-or-later
- **Copyleft:** Yes - modifications must be shared
- **Commercial:** Allowed with AGPL compliance
- **Network:** AGPL applies to network use
- **Patent:** Grant included

### Why AGPL-3.0?
- Strong copyleft ensures contributions stay open
- Network use protection for SaaS deployments
- Enterprise adoption with compliance
- Community protection from proprietary forks

## Commercial License

### Enterprise License
- **Type:** Proprietary
- **Features:** Enterprise features
- **Support:** Dedicated support
- **Terms:** Annual subscription

### Enterprise Plus License
- **Type:** Proprietary
- **Features:** Custom development
- **Support:** 24/7, on-site
- **Terms:** Custom contract
