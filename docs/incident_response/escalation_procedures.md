# Escalation Procedures

## Automatic Escalation

| Time Since Detection | Action |
|:--------------------:|--------|
| 0 min | Alert on-call engineer |
| 5 min | If unacknowledged, page backup |
| 15 min | If unresolved, escalate to Engineering Manager |
| 30 min | If unresolved, escalate to VP Engineering |
| 1 hour | If unresolved, escalate to CTO |
| 2 hours | If unresolved, activate all-hands |

## Manual Escalation

### To Engineering Manager
1. Notify via Slack #incidents channel
2. Tag @engineering-manager
3. Provide incident summary

### To VP Engineering
1. Call VP Engineering directly
2. Provide incident summary
3. Request immediate support

### To CTO
1. Call CTO directly
2. Provide incident summary
3. Request executive decision

## Communication Channels

| Channel | Purpose | Participants |
|---------|---------|--------------|
| #incidents | Real-time updates | All engineering |
| Incident Bridge | War room | Response team |
| Executive Update | Status updates | Leadership |
| Customer Update | External comms | Support, Marketing |
