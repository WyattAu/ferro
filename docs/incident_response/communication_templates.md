# Communication Templates

## Initial Detection

**Internal:**
```
INCIDENT DETECTED - Level [1/2/3/4]

Summary: [Brief description]
Impact: [User impact]
Start Time: [Timestamp]
On-Call: [Engineer name]

Status: Investigating
Next Update: [Time]
```

**External:**
```
We're currently investigating an issue affecting [feature/service].

Impact: [Description of user impact]
Start Time: [Timestamp]

We're working to resolve this as quickly as possible. Updates will be posted here.
```

## Status Update

**Internal:**
```
INCIDENT UPDATE - Level [1/2/3/4]

Summary: [Brief description]
Current Status: [Investigating/Mitigating/Resolved]
Root Cause: [If known]
Actions Taken: [List of actions]
Next Steps: [Planned actions]
Next Update: [Time]
```

**External:**
```
Update on current issue:

Status: [Investigating/Mitigating/Resolved]
Impact: [Current user impact]
Actions: [What we're doing]

We'll continue to provide updates every [frequency].
```

## Resolution

**Internal:**
```
INCIDENT RESOLVED - Level [1/2/3/4]

Summary: [Brief description]
Duration: [Total time]
Root Cause: [Root cause]
Resolution: [How it was resolved]
Impact: [Final impact assessment]

Post-mortem scheduled for [date/time].
```

**External:**
```
The issue affecting [feature/service] has been resolved.

Duration: [Total time]
Root Cause: [Root cause]
Resolution: [How it was resolved]

We apologize for the inconvenience. A post-mortem will be conducted to prevent similar issues in the future.
```
