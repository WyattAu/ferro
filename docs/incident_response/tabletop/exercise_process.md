# Tabletop Exercise Process

## Pre-Exercise (1 week before)

- [ ] Confirm participants
- [ ] Distribute scenario materials
- [ ] Schedule meeting room
- [ ] Prepare facilitator notes

## Exercise Execution

### Opening (5 minutes)
1. Welcome participants
2. Review objectives
3. Set ground rules

### Scenario Introduction (5 minutes)
1. Present scenario
2. Share timeline
3. Answer initial questions

### Initial Response (10 minutes)
1. Participants discuss initial actions
2. Identify roles and responsibilities
3. Establish communication channels

### Triage and Assessment (15 minutes)
1. Assess impact and severity
2. Identify root cause
3. Determine escalation needs

### Mitigation Actions (15 minutes)
1. Discuss mitigation strategies
2. Assign actions
3. Practice communication

### Recovery Planning (10 minutes)
1. Plan recovery steps
2. Identify dependencies
3. Set timeline

### Debrief (5 minutes)
1. Review lessons learned
2. Identify action items
3. Schedule follow-up

## Post-Exercise (1 week after)

- [ ] Distribute after-action report
- [ ] Track action items
- [ ] Update runbooks
- [ ] Schedule next exercise
