# Scenario 2: Security Breach

## Scenario Description

At 10:00 AM, security team detects unauthorized access to production systems. Logs show suspicious activity from an unknown IP address. Customer data may have been accessed.

## Timeline

| Time | Event |
|------|-------|
| 10:00 AM | Alert triggered: Suspicious activity detected |
| 10:05 AM | Security team acknowledges alert |
| 10:10 AM | Investigation begins |
| 10:15 AM | Unauthorized access confirmed |
| 10:20 AM | Systems isolated |
| 10:25 AM | Forensic investigation begins |
| 10:30 AM | Customer notification prepared |

## Discussion Points

### Detection
- How did we detect the breach?

### Response
- Did we follow the security incident runbook?
- Was the escalation appropriate?
- Did we preserve evidence?

### Recovery
- How did we contain the breach?
- Was customer notification appropriate?
- What improvements are needed?
