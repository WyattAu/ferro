# Ferro Community Events

## Overview

Ferro hosts regular community events to connect contributors and users.

## Regular Events

### Monthly Meetup
- **When:** First Wednesday of each month
- **Where:** Discord
- **Duration:** 1 hour
- **Agenda:**
  - Welcome and introductions
  - Project updates
  - Technical deep dive
  - Open discussion
  - Q&A

### Quarterly Release Party
- **When:** After each quarterly release
- **Where:** Discord + Twitter Space
- **Duration:** 2 hours
- **Agenda:**
  - Release highlights
  - Demo of new features
  - Contributor recognition
  - Community feedback
  - Roadmap preview

### Annual Conference
- **When:** October (Ferropalooza)
- **Where:** Virtual + In-person (location TBD)
- **Duration:** 2 days
- **Agenda:**
  - Keynote
  - Technical talks
  - Workshops
  - Networking
  - Contributor awards

## Special Events

### Hacktoberfest
- **When:** October
- **Where:** GitHub
- **Duration:** Month-long
- **Activities:**
  - Beginner-friendly issues
  - Mentorship
  - Swag for contributors

### Advent of Code
- **When:** December
- **Where:** GitHub
- **Duration:** 25 days
- **Activities:**
  - Daily challenges
  - Community solutions
  - Leaderboard

### Google Summer of Code
- **When:** May-August
- **Where:** GitHub
- **Duration:** 3 months
- **Activities:**
  - Student projects
  - Mentorship
  - Stipends

## Contributing to Events

### Proposing an Event
1. Create GitHub issue
2. Describe event
3. Get community feedback
4. Find organizers
5. Schedule event

### Organizing an Event
1. Create event page
2. Promote on social media
3. Prepare materials
4. Run event
5. Follow up

### Speaking at an Event
1. Submit proposal
2. Prepare presentation
3. Deliver presentation
4. Answer questions
5. Share slides
