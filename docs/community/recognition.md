# Ferro Community Recognition

## Overview

Ferro recognizes and rewards community contributions.

## Contributor Levels

### Bronze Contributor
- 1-5 merged PRs
- Badge on GitHub profile
- Access to contributor Discord channel

### Silver Contributor
- 6-20 merged PRs
- Badge on GitHub profile
- Access to contributor Discord channel
- Swag (stickers, t-shirt)

### Gold Contributor
- 21-50 merged PRs
- Badge on GitHub profile
- Access to contributor Discord channel
- Swag (stickers, t-shirt, hoodie)
- Name in CONTRIBUTORS.md

### Platinum Contributor
- 51+ merged PRs
- Badge on GitHub profile
- Access to contributor Discord channel
- Swag (stickers, t-shirt, hoodie, jacket)
- Name in CONTRIBUTORS.md
- Invitation to maintainer team
