## Detailed Migration Plans

### 1. Tauri Plugin Integration

#### `tauri-plugin-context-menu`

**Current State:** No native context menus. Web UI uses custom right-click handling.

**Migration:**
```rust
// In crates/desktop/src/gui.rs
use tauri_plugin_context_menu::ContextMenuExt;

// Register context menu for file browser
app.context_menu().add_menu_item("Open", || { /* ... */ });
app.context_menu().add_menu_item("Delete", || { /* ... */ });
app.context_menu().add_separator();
app.context_menu().add_menu_item("Properties", || { /* ... */ });
```

**Effort:** 2-3 days
**Risk:** Low - additive change

#### `tauri-plugin-fs-pro`

**Current State:** Basic file operations via Tauri commands.

**Migration:**
```rust
// Enhanced file operations
use tauri_plugin_fs_pro::FsProExt;

// File picker with filters
app.fs_pro().open_file picker()
    .add_filter("Documents", &["pdf", "doc", "docx"])
    .pick_file(|path| { /* ... */ });

// Directory picker
app.fs_pro().pick_folder(|path| { /* ... */ });
```

**Effort:** 3-4 days
**Risk:** Medium - may require restructuring file operations

#### `sentry-tauri`

**Current State:** No error tracking.

**Migration:**
```rust
// In crates/desktop/src/main.rs
use sentry_tauri::SentryExt;

app.setup(|app| {
    let _guard = sentry_tauri::init(sentry_tauri::SentryOptions {
        dsn: "https://your-dsn@sentry.io/project-id".into(),
        release: sentry_tauri::release_name!(),
        ..Default::default()
    });
    Ok(())
});
```

**Effort:** 1-2 days
**Risk:** Low - additive change

### 2. Leptos Package Integration

#### `leptos-fetch`

**Current State:** Custom API fetching in each component.

**Migration:**
```rust
// Replace custom fetch with leptos-fetch
use leptos_fetch::Query;

#[component]
pub fn FileList() -> impl IntoView {
    let files = Query::new(|| async {
        api.list_files().await
    })
    .cache_time(std::time::Duration::from_secs(30))
    .build();

    view! {
        {move || files.get().map(|data| view! {
            // Render files
        })}
    }
}
```

**Effort:** 5-7 days (refactor all API calls)
**Risk:** Medium - affects all data-fetching components

#### `leptos-use`

**Current State:** Custom reactive hooks.

**Migration:**
```rust
// Replace custom hooks with leptos-use
use leptos_use::{use_debounce_fn, use_intersection_observer};

#[component]
pub fn SearchInput() -> impl IntoView {
    let (query, set_query) = create_signal(String::new());
    let debounced_query = use_debounce_fn(move || query.get(), 300);

    view! {
        <input
            type="text"
            prop:value=query
            on:input=move |ev| set_query.set(event_target_value(&ev))
        />
    }
}
```

**Effort:** 3-4 days
**Risk:** Low - drop-in replacement

#### `leptix`

**Current State:** Custom accessible components.

**Migration:**
```rust
// Replace custom dialog with leptix
use leptix::dialog::{Dialog, DialogTrigger, DialogContent};

view! {
    <Dialog>
        <DialogTrigger>"Open Dialog"</DialogTrigger>
        <DialogContent>
            // Dialog content
        </DialogContent>

```

**Effort:** 4-5 days
**Risk:** Medium - may require design system adjustments

#### `leptos-struct-table`

**Current State:** Custom table implementation in admin UI.

**Migration:**
```rust
// Replace custom table with leptos-struct-table
use leptos_struct_table::{Table, TableColumn};

#[derive(Table)]
#[table(optimize)]
struct User {
    #[table(name = "ID")]
    id: String,
    #[table(name = "Username")]
    username: String,
    #[table(name = "Email")]
    email: String,
}

#[component]
pub fn UserTable() -> impl IntoView {
    let users = create_resource(|| (), |_| async { api.list_users().await });

    view! {
        <Table rows=users column=column_configuration />
    }
}
```

**Effort:** 3-4 days
**Risk:** Medium - significant refactor of admin tables

## Recommendation

**Adopt incrementally.** Start with high-value, low-risk items:

1. **Week 1-2:** Adopt `tauri-plugin-context-menu`, `tauri-plugin-fs-pro`, `sentry-tauri`, `leptos-fetch`, `leptos-use`
2. **Week 3-4:** Adopt `leptix`, `leptos-struct-table`, `leptos-chartistry`, `lepticons`
3. **Week 5-6:** Adopt `Stylance`, `Tailwind Fuse`, `leptos_animate`, `tauri-plugin-theme`, `leptos-hotkeys`
4. **Week 7-8:** Evaluate `Thaw`, `Rust shadcn/ui`, `leptos-darkmode`

**Do NOT migrate everything at once.** Each package should be evaluated individually for compatibility with Ferro's design system.
