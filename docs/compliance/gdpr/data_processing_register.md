# Data Processing Register

## Processing Activities

### User Account Data
- **Purpose:** User authentication and account management
- **Legal Basis:** Contract performance (Article 6(1)(b))
- **Data Categories:** Name, email, password hash, TOTP secret
- **Retention:** Account lifetime + 30 days
- **Security:** Encryption at rest, access controls

### Calendar Data
- **Purpose:** Calendar management and synchronization
- **Legal Basis:** Contract performance (Article 6(1)(b))
- **Data Categories:** Events, attendees, locations, descriptions
- **Retention:** Account lifetime + 30 days
- **Security:** Encryption at rest, access controls

### Contact Data
- **Purpose:** Contact management and synchronization
- **Legal Basis:** Contract performance (Article 6(1)(b))
- **Data Categories:** Names, emails, phone numbers, addresses
- **Retention:** Account lifetime + 30 days
- **Security:** Encryption at rest, access controls

### Audit Logs
- **Purpose:** Security monitoring and compliance
- **Legal Basis:** Legitimate interest (Article 6(1)(f))
- **Data Categories:** User actions, IP addresses, timestamps
- **Retention:** 12 months
- **Security:** Immutable storage, access controls
