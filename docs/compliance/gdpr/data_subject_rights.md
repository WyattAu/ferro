# Data Subject Rights

## Right of Access (Article 15)

### Process
1. User submits access request
2. System verifies identity
3. Data compilation (30 days)
4. Response provided to user

### Data Included
- Personal data processed
- Processing purposes
- Recipients
- Retention periods
- Source of data

## Right to Rectification (Article 16)

### Process
1. User submits rectification request
2. System verifies identity
3. Data updated
4. Confirmation sent to user

## Right to Erasure (Article 17)

### Process
1. User submits erasure request
2. System verifies identity
3. Legal obligations checked
4. Data deleted (30 days)
5. Confirmation sent to user

### Exceptions
- Legal obligations
- Public interest
- Legal claims

## Right to Data Portability (Article 20)

### Process
1. User submits portability request
2. System verifies identity
3. Data exported in machine-readable format
4. Data provided to user

### Formats
- JSON
- CSV
- vCard
- iCalendar
