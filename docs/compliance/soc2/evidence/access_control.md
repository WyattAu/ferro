# Access Control Evidence

## MFA Implementation

### Evidence
- MFA configuration in Okta/Auth0
- User MFA enrollment logs
- MFA bypass requests (if any)

### Collection Method
- Automated: Okta API export
- Manual: Screenshot of MFA settings

### Retention
- 12 months

## RBAC Implementation

### Evidence
- Role definitions document
- User-role assignments
- Access review logs

### Collection Method
- Automated: IAM export
- Manual: Review meeting notes

### Retention
- 12 months

## Access Reviews

### Evidence
- Quarterly access review reports
- Review meeting notes
- Remediation actions

### Collection Method
- Manual: Review process
- Automated: IAM export

### Retention
- 12 months
